# Prisma+MongoDB Lab

## Tech stacks

- Express JS
- Prisma
- Prisma Client
- Json Web Token
- Cookie Parser
- Bcrypt JS
- Cors
- Dotenv

## Setup

1. Install dependencies:

```zsh
pnpm i
```

2. Push your schema to MongoDB

```zsh
pnpx prisma@6 db push
```

3. Generate prisma client

```zsh
pnpx prisma@6 generate
```
