# Auth service

## Tech Stack

### Core:

- Runtime: Node.js
- Framework: Fastify (v5.x)
- Language: TypeScript (v6.x)
- Database: MongoDB

### Security & Auth:

- Hashing: Argon2
- Tokens: JWT (JSON Web Tokens)
- Storage: Signed HttpOnly Cookies

### Documentation:

- Swagger: OpenAPI 3.0 via src/docs/openapi.yaml

## Prerequisites

- Node.js: v20+
- Package Manager: pnpm (v10+)
- Database: A running MongoDB instance
